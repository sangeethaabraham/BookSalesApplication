Java Assignment - Book Sales
=============================

Java Version used : Java 1.8
Development Environment : Eclipse Mars
Tested on Windows version only.

To Setup Project:
=================
--Extract Source files.
--Ensure csv file paths are given correctly as input. 
--In this assignment csv files are kept in resources folder.
--Import project into Eclipse.

To set arguments:
===================
-----Right Click and select BookSales.java->Run As->Run Configuration
-----Select Arguments Tab and set the sample argument list as below.
--Sample Argument List is :
--books=./src/resources/books.list --sales=./src/resources/sales.list --top_selling_books=2 --top_customers=2 --sales_on_date=2018-08-01

To Run Application
==================
----->Right Click and select BookSales.java->Run As->Java Application
